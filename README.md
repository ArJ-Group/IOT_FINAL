# Smart Plant Watering 

This system is automatically water the plant when the soil is dry. The data is monitored in dashboard using Node Red

**Device used:** 
- NodeMCU ESP8266
- Soil moisture sensor
- DHT 11
- 9v water pump
- Relay 
- 9v Battery
- LED
